1 创建 GitHub 仓库

方法一：网页上传（最快）

  1. 打开 github.com/new
  2. Repository name 填 etsy688-local，选 Private，点击 Create
  3. 进入仓库 → Add file → Upload files
  4. 把 etsy688-local 文件夹里的所有内容拖进去
    （不需要 node_modules，Railway 会自己安装）
  5. 点击 Commit changes

    6. 方法二：Git 命令行

    cd etsy688-local rm -rf node_modules git init && git add . && git commit -m "Etsy688 Local v1.0" git branch -M main git remote add origin https://github.com/YOUR_USERNAME/etsy688-local.git git push -u origin main

2 连接 Railway

  1. 打开 railway.app，用 GitHub 账号登录
  2. 点击 New Project → Deploy from GitHub repo
  3. 搜索并选择 etsy688-local 仓库
  4. Railway 自动识别为 Node.js 项目（已有 package.json + Procfile）

3 配置环境变量

在 Railway 项目面板 → Variables，添加：

变量名值说明
PORT3001服务端口
NODE_ENVproduction生产模式
REPLICATE_TOKENr8_xxx...激活AI图片功能
OPENAI_API_KEYsk-xxx...激活SEO功能
ADMIN_PIN1234管理面板PIN
⚠️ 不配置Key也能部署
尺寸标注功能纯前端实现，无需Key即可使用。AI图片和SEO需要Key。

4 部署 & 验证

  1. 点击 Deploy Now，等待构建（约2-3分钟）
  2. 构建成功后显示公共 URL：https://etsy688-local.up.railway.app
  3. 打开验证：主界面 + 配置面板（/admin）
