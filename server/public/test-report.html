📋 功能模块说明

📷
主生成器
上传产品图，生成商业场景图，支持智能匹配/真人模特/家居/Pinterest/Instagram/自定义风格
模型：SDXL via Replicate
🎬
视频创作
上传产品图，AI 生成动态商品展示视频，支持5秒/10秒时长
模型：Zeroscope v2 via Replicate
👨
模特场景
白底图→模特穿戴展示图，支持户外/咖啡馆/卧室/街拍/影棚/自定义场景
模型：SDXL via Replicate
🔍
质感增强
AI 超分辨率放大，2x/4x倍，增强图片清晰度和质感
模型：Real-ESRGAN via Replicate
🕑
尺寸标注
输入 L×W×H (cm/inch)，Canvas 自动标注技术尺寸线，无需AI
纯前端 Canvas 实现
🧪
创意合成
最多4张产品图融合到同一商业场景，支持自定义合成描述
模型：SDXL via Replicate
🔍
SEO 工具
输入关键词，生成 Etsy SEO 优化标题/描述/Tags，支持关键词生成模式
模型：GPT-4o-mini via OpenAI
⚙
配置面板
PIN保护（默认1234），统一管理 Replicate / OpenAI API Key，重启生效
安全：Key存在服务端，非前端

⚠️ 正式使用前需要

AI 功能需要配置 API Key（免费）：
  1. Replicate（推荐，支持SDXL+视频）：
    注册 replicate.com → Account API Tokens
    最低成本约 $0.002/张图片
  2. OpenAI（SEO功能）：
    注册 platform.openai.com → API Keys
    GPT-4o-mini 成本极低，约 $0.01/次SEO生成
配置路径：访问 /admin → 输入 PIN 1234 → 填入 Key → 保存 → 重启服务
